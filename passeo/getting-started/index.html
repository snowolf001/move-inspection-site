Getting Started

Start by setting up your club, inviting members, and preparing your first session. Passeo is designed to keep the setup simple so you can begin using it quickly and add more structure over time.

View Roles View Sessions

Basic setup flow

A simple setup path helps clubs get started without unnecessary complexity.

1

Create your club

Set up the club with a name and the basic structure you want members to use.

2

Invite members

Add members directly or share a join method so people can join the club more easily.

3

Assign roles

Decide who should be owner, admin, host, or member based on how your club operates.

4

Create your first session

Add a session so members can join, check in, and begin using the app in a real workflow.

What to set up first

Focus on the essentials first, then refine your process after your club begins using the app.

• Club name and basic club identity
• Member join method or join code flow
• Role assignments for organizers
• Session creation workflow
• Attendance and participation habits

Helpful tips

Keep the first setup simple

Do not try to define every rule at the beginning. Start with the essentials and improve the workflow after real use.

Use roles clearly

Clear role boundaries make it easier to manage sessions, attendance, and member actions without confusion.

Test one real session

Running one real or trial session is often the fastest way to confirm that your setup works for your club.
