Beginner-friendly guide

How to Start with Passeo

Set up your club, create sessions, add credits, and start check-ins in just a few simple steps.

Get it on Google Play Download on the App Store View Passeo Features

Start Simple

You do not need to set up everything perfectly on day one.

Many clubs begin with:

  • One session
  • A few members
  • Simple manual check-ins

Passeo is designed to help clubs gradually move away from paper cards and manual attendance tracking.

Step-by-step setup guide

Follow these steps to set up your club and run your first session with Passeo.

1

Create Your Club

Create your club and become the owner automatically. This gives you full control over club settings and management.

  • Choose a club name
  • Select your sport or activity type
  • Invite members later when ready
2

Create Your First Session

Create a session with date, time, and optional player capacity. Sessions can be one-time or recurring.

  • Recurring sessions supported
  • Optional location details
  • Optional capacity limits
3

Invite Members

Share your club join code so members can join themselves. No manual setup needed for each person.

  • Members install the app
  • Members join using the code
  • Less manual management for you
4

Add Credits

Managers can add credits to members before sessions. This helps track participation and manage attendance.

  • Flexible credit amounts
  • Support for guests
  • Easy adjustments later
5

Start Check-In

Members can check themselves in, or hosts can manually check in members. Choose what works best for your club.

  • Live attendance updates
  • Manual backup available
  • Useful for busy sessions
6

Review Attendance

Review attendance history and participation after sessions. Track who attended and when.

  • See who attended
  • View participation history
  • Track club activity

Common Questions

Questions club managers often have when starting with Passeo.

Do all members need the app?

No. Managers and hosts can still manually check in members or guests who do not have the app installed.

What if someone forgets to check in?

Hosts or managers can manually check them in later. The system is flexible and allows for corrections.

Can I still use paper cards?

Yes. Many clubs transition gradually while learning the system. Use whatever combination works for your club.

What if I make a mistake?

Credits and attendance can be adjusted later. You have control to fix any errors that occur.

Is Passeo only for large clubs?

No. Passeo works well for small groups and large clubs. Start with any size and grow over time.

How do recurring sessions work?

Create a session template that repeats weekly, biweekly, or monthly. Save time by not recreating the same session.

Ready to Try Passeo?

Create your club and run your first session in minutes. No long setup or complicated configuration required.
