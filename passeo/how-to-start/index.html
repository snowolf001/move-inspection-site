Step-by-step guide

How to Start with Passeo

Follow these steps to set up your club and run your first session.

Get it on Google Play Download on the App Store View Passeo Features

Start Simple

You do not need to set up everything perfectly on day one.

Most clubs begin with:

  • One session
  • A few members
  • Simple manual check-ins

Passeo is designed to help clubs gradually move away from paper cards and manual attendance tracking.

Step 1

Create Your Club

Instructions:

  1. Open Passeo
  2. Tap "Create Club"
  3. Enter:
    • First Name
    • Last Name
    • Club Name
  4. Tap Create

Important notes:

  • You automatically become the club owner
  • Owners can manage the club and promote hosts later
Screenshot: create-club.png

Create your club in less than a minute.

Step 2

Create Your First Session

Sessions are where members check in and attendance is tracked.

Instructions:

  1. Open your club Home screen
  2. Tap the "New Session" button
  3. On the New Session page:
    • Select a location
    • Select a host, or leave it empty for no host
    • Enter a session name
      • If left blank, the location name will be used automatically
    • Choose the session date
    • Choose the start time
    • Choose the end time
  4. Tap "Create Session"

Important notes:

  • Sessions are used for attendance and check-ins
  • Hosts can help manage check-in during the session
  • Player capacity is optional if supported by the selected location
Screenshot: home-new-session-button.png

Create sessions directly from the club home screen.

Screenshot: new-session-page.png

Set location, host, and session time in just a few taps.

Step 3

Invite Members

Instructions:

  1. Open Club Settings
  2. Find your Club Join Code
  3. Share the code with members
  4. Members:
    • Install Passeo
    • Tap "Join Club"
    • Enter the join code

Important notes:

  • Members join by themselves
  • No need to manually create every member
Screenshot: club-join-code.png

Members can quickly join using your club join code.

Step 4

Promote a Member to Host

Instructions:

  1. Open Profile
  2. Tap "Manage Members"
  3. Select a member
  4. Tap "Promote to Host"

Important notes:

  • Only the club owner can promote hosts
  • Hosts can help manage sessions and check-ins
Screenshot: promote-host.png

Hosts can help manage attendance and sessions.

Step 5

Add Credits to Members

Credits represent how many sessions a member can attend.

Example: 1 session = 1 credit

Instructions:

  1. Open Profile
  2. Tap "Manage Members"
  3. Select a member
  4. Tap "Adjust Credits"
  5. Enter the number of credits
  6. Tap Save

Important notes:

  • Credits can be adjusted later
  • Clubs can choose their own credit system
Screenshot: manage-members.png

Add credits so members can check in to sessions.

Optional Feature

See Who's Going

Members can mark whether they plan to attend before check-in starts.

"Who's Going" helps estimate attendance before the session. Check-in records actual attendance and credit usage during the session.

Instructions:

  1. Members open a session
  2. Tap "Going"

Important notes:

  • Useful for estimating attendance
  • Helps clubs prepare enough courts/tables/space
  • Separate from actual check-in
Screenshot: whos-going-session.png

See expected attendance before sessions begin.

Step 6

Start Check-In

Instructions:

  1. Open a session
  2. Tap "Check-In Mode"
  3. Members can check themselves in
  4. Hosts can also manually check in members

Important notes:

  • During check-in, you can choose how many credits to use
  • Example: normal session = 1 credit, guest/player fee could use more credits
  • Manual check-in helps when members forget their phones
Screenshot: check-in-mode.png

Track attendance live during each session.

Step 7

Review Attendance

Instructions:

  1. Open a session
  2. View checked-in members
  3. Open attendance history or reports

Important notes:

  • Review who attended
  • Track participation history
  • See club activity over time
Screenshot: attendance-history.png

Review attendance after every session.

Common Questions

Answers to questions club managers often have.

Do all members need the app?

No. Hosts and managers can still manually check in members or guests.

What if someone forgets to check in?

Hosts or managers can manually check them in later.

Can I still use paper cards at first?

Yes. Many clubs transition gradually while learning the system.

What if I make a mistake?

Credits and attendance can be adjusted later.

Is Passeo only for large clubs?

No. Passeo works well for both small groups and large clubs.
